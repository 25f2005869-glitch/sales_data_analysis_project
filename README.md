📊 Sales Data Analysis Project

📌 Overview

This project analyzes monthly sales data of three products (Product A, Product B, Product C) using Python.
It demonstrates data analysis and visualization using Pandas and Matplotlib.

---

📁 Project Structure

sales_data_analysis_project/
│── sales_data.csv
│── sales_data_analysis.py
│── sales_line_chart.png
│── sales_bar_chart.png
│── sales_distribution_may.png
│── README.md

---

📊 Features

- 📈 Line Chart → Monthly sales comparison
- 📊 Bar Chart → Product-wise comparison
- 🥧 Pie Chart → Sales distribution (May)
- 📂 CSV Data Handling using Pandas

---

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib

---

📷 Output Graphs

📈 Line Chart

Shows monthly sales comparison of all products.

📊 Bar Chart

Displays product-wise sales in each month.

🥧 Pie Chart

Represents percentage contribution of each product in May.

---

▶️ How to Run

1. Install required libraries:

pip install pandas matplotlib

2. Run the script:

python sales_data_analysis.py
